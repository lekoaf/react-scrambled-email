# react-scrambled-email

### Installation

`yarn add react-scrambled-email`

### Usage

```jsx
import React from 'react'
import { ScambledEmail } from 'react-scrambled-email'

export const App = () => {
  return (
    <span>
      Send me an
      <ScrambledEmail email="me@myemail.com" text="email" />
    </span>
  )
}
```
